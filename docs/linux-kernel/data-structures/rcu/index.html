    RCU Read-Copy-Update
    • 1. 原理
    • 2. 关键思想

    1. 原理 #

    • 对一块数据要写入时，先拷贝一份，更新数据完成后，再更新到原数据结构
    • 保证要么读取老数据，要么读取新数据，不会出现读取一半的情况

    2. 关键思想 #

    • 复制后更新数据
    • 延迟回收内存，当没有老数据结构引用的读者时，才回收内存
